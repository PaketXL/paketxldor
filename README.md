

# How to run with TERMUX
1. Update & Upgrade Termux
```
pkg update && pkg upgrade -y
```
2. Install Git
```
pkg install git -y
```
3. Clone this repo
```
git clone https://github.com/HackersXl/pakeetxl
```
4. Open the folder
```
cd pakeetxl
```
5. Setup
```
bash setup.sh
```
6. Run the script
```
python main.py
```
7. Shell
```
git pull --rebase
```
8. Reset Shell
```
git reset --hard
```
